DeepChem

Description

DeepChem is an open-source toolkit for deep learning in drug discovery, materials science, and chemistry, providing tools for molecular modeling and property prediction.

Company & History

DeepChem was launched in 2016 as a community-driven project to bring advanced machine learning to chemistry and life sciences.

Versions & Plans

  • Open Source: Free and community-supported.

How to Use

  • Visit deepchem.io to get started.
  • Install DeepChem via pip or conda.
  • Explore tutorials and documentation for molecular modeling tasks.
  • Contribute to the open-source community.

Key Features

  • Deep learning for chemistry
  • Molecular property prediction
  • Drug discovery tools
  • Active open-source community

Pricing

  • Open Source: Free
