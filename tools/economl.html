EconoML

Description

EconoML is an open-source Python package for estimating causal effects in economics and data science, using machine learning for economic modeling and inference.

Company & History

EconoML is developed by Microsoft Research and the open-source community to advance causal inference and econometric analysis with AI.

Versions & Plans

  • Open Source: Free and community-supported.

How to Use

  • Visit econml.azurewebsites.net or the GitHub repo.
  • Install via pip: pip install econml
  • Follow documentation and tutorials for causal inference tasks.

Key Features

  • Causal inference with machine learning
  • Open-source and extensible
  • Python API and documentation
  • Active research community

Pricing

  • Open Source: Free
