Unity ML-Agents

Description

Unity ML-Agents is an open-source toolkit for training intelligent agents using reinforcement learning, imitation learning, and other machine learning techniques in Unity games.

Company & History

Developed by Unity Technologies, ML-Agents is widely used in both research and commercial game development for creating smart, adaptive game behaviors.

Versions & Plans

  • Open Source: Free and community-supported.
  • Enterprise: Commercial support and advanced features.

How to Use

  • Visit Unity ML-Agents to get started.
  • Install the ML-Agents toolkit in your Unity project.
  • Define agents and training environments.
  • Train agents using Python API and integrate into your game.

Key Features

  • Reinforcement and imitation learning
  • Integration with Unity Editor
  • Flexible agent and environment setup
  • Open-source and extensible

Pricing

  • Open Source: Free
  • Enterprise: Contact Unity for pricing
